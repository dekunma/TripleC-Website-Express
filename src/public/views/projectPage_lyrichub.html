LyricHub Project
Description Demo Timeline Team

Project Description

Our project aimed at taking intensive analysis on 380,000+ popular singers’ lyrics. With the underlying sentiment analysis interface, LyricHub would be able to visualize popular word choices of each singer and providing the tangible explanation of such word choice. Furthermore, we were doing nuanced analysis in determining the possible connection between word choice and popularity of the song and thus providing popularity prediction with any lyrics given.
